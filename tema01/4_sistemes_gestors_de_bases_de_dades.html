4. Sistemes Gestors de Bases de Dades

Sistema Gestor de Base de dades (SGBD o DBMS): conjunt coordinat de programes, procediments, llenguatges, etc., que subministra, tant als usuaris no informàtics, com als analistes, programadors, o a l'administrador, els mitjans necessaris per a descriure i manipular les dades contingudes en la base de dades, mantenint la seua integritat, confidencialitat i seguretat.

Funcions

Un SGBD desenvolupa tres funcions fonamentals:

  • Funció de descripció o definició: permet al dissenyador de la base de dades crear les estructures apropiades per a integrar adequadament les dades. Aquesta funció es realitza mitjançant el llenguatge de descripció de dades o DDL.

  • Funció de manipulació: permet als usuaris de la base buscar, afegir, suprimir o modificar les dades d'aquesta, sempre d'acord amb les especificacions i les normes de seguretat dictades per l'administrador. Es durà a terme per mitjà d'un llenguatge de manipulació de dades ( DML) que facilita els instruments necessaris per a la realització d'aquestes tasques.

  • Funció de control: permet a l'administrador de la base de dades establir mecanismes de protecció de les diferents visions de les dades associades a cada usuari, proporcionant elements de creació i modificació d'aquests usuaris. Addicionalment, incorpora sistemes per a la creació de còpies de seguretat, càrrega de fitxers, auditoria, protecció d'atacs, configuració del sistema, etc. El llenguatge que implementa aquesta funció és el llenguatge de control de dades o DCL.

Components

Els SGBD solen estar formats per diversos components integrats que fan tasques de gestió de dades:

  • Motor d'emmagatzematge: és l'element central d'un SGBD que interactua amb el sistema d'arxius nivell del sistema operatiu.

  • Llenguatge de consulta: permeten als usuaris interactuar amb les bases de dades (exemple: SQL o MQL).

  • Processador de consultes: interpreta les consultes de l'usuari i fa que els comandos siguen comprensibles per a la base de dades.

  • Optimizador de consultes: proporciona idees per a millorar el rendiment de la base de dades i les consultes.

  • Diccionari de Dades: emmagatzema totes les metadades generades per una entitat, així com informació sobre l'estructura de la BBDD. El diccionari s'utilitza per a verificar les sol·licituds dels usuaris i proporcionar detalls sobre l'estructura de la base de dades.

  • Administrador de registres: registra l'activitat dels usuaris, els inicis de sessió, les còpies de seguretat i altres accions realitzades amb el SGBD.

  • Eines d'informes i monitoratge: generen informes i supervisen l'ús dels recursos del SGBD.

  • Utilitats de dades: eines addicionals que fan tasques de còpia de seguretat i recuperació, validació de dades, reparació de bases de dades i comprovació de la integritat de les dades en el teu servidor privat o compartit.

Tipus

Vist en l'apartat anterior de tipus de BBDD és aplicable als SGBD.
